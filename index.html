Proyectos

1. Simulación de Convergencia Probabilística con Streamlit

Contexto del proyecto:
Desarrollo de una aplicación web para simular lanzamientos de una moneda utilizando una distribución Bernoulli. El objetivo fue visualizar cómo la media converge hacia 0.5 conforme aumenta el número de iteraciones, ilustrando la Ley de los Grandes Números.

Análisis:
Se utilizó SciPy para generar resultados aleatorios y calcular la proporción acumulada de éxitos. La aplicación fue desarrollada en Streamlit, incorporando control de estado con session_state, visualización dinámica mediante gráficos de línea y almacenamiento de resultados en un DataFrame para registrar múltiples experimentos.

Conclusiones principales:
A medida que aumenta el número de intentos, la media de los resultados tiende a estabilizarse alrededor de 0.5, confirmando el comportamiento esperado en eventos aleatorios independientes.

2. Aplicación Web para Análisis Exploratorio de Vehículos en Venta

Contexto del proyecto:
Construcción de una aplicación web interactiva para explorar un conjunto de datos de anuncios de venta de vehículos en Estados Unidos.

Análisis:
Se utilizó Pandas para la manipulación del dataset y Plotly Express para la generación de visualizaciones interactivas. La aplicación permite construir histogramas del odómetro y gráficos de dispersión entre precio y kilometraje mediante botones en Streamlit.

Conclusiones principales:
El análisis permitió observar la distribución del kilometraje y la relación entre precio y odómetro, evidenciando cómo el uso del vehículo influye en su valor de mercado.
